Usage Examples

Basic Usage

# Single file
pydeadcode app.py

# Multiple files
pydeadcode file1.py file2.py

# All files in directory
pydeadcode src/**/*.py

Advanced Options

# Set confidence threshold
pydeadcode main.py --min-confidence 75

# JSON output for CI/CD
pydeadcode main.py --json > report.json

# Verbose output
pydeadcode main.py --verbose
